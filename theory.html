<!DOCTYPE> (Document Type Declaration) використовується в HTML для вказівки типу документа та версії мови HTML, яку слід використовувати під час відображення веб-сторінки браузером. Це необхідно для того, щоб браузер правильно інтерпретував HTML-код сторінки.

Приклад коректного <!DOCTYPE> для HTML5 виглядає так:

<!DOCTYPE html>

Цей рядок вказує браузеру те що, що документ є HTML5-документом. <!DOCTYPE html> має бути першим рядком у HTML-коді перед відкриваючим тегом <html>.

У HTML створення нумерованих та маркованих списків здійснюється з використанням тегів <ol> (ordered list) та <ul> (unordered list) відповідно. Нумеровані списки використовуються для впорядкованих елементів, а марковані списки для невпорядкованих.

Приклад нумерованого списку:

<ol>
<li>Перший елемент</li>
<li>Другий елемент</li>
<li>Третій елемент</li>
</ol>


Приклад маркованого списку:

<ul>
<li>Перший елемент</li>
<li>Другий елемент</li>
<li>Третій елемент</li>
</ul>


Щоб змінити порядок нумерації у списку, можна використовувати атрибут start для тега <ol>. Наприклад, щоб почати нумерацію з певного числа:

<ol start="5">
<li>П'ятий елемент</li>
<li>Шостий елемент</li>
<li>Сьомий елемент</li>
</ol>


У цьому випадку нумерація починається з числа 5.

Для забезпечення переходу до іншого розділу поточного документа HTML використовуються якорі (anchors) і відповідні їм гіперпосилання. Якір - це точка всередині документа, до якої можна створити посилання переходу. Щоб створити якір, використовується тег <a> (гіперпосилання) з атрибутом name або id, і цей якір можна використовувати як ціль для гіперпосилань.

Приклад створення якоря:

<a id="section1"></a>
<h2>Розділ 1</h2>
<p>Вміст розділу 1</p>


Приклад гіперпосилання на якір:

<a href="#section1">Перейти до Розділу 1</a>


Тепер про різницю між абсолютними та відносними гіперпосиланнями:

  1. Абсолютні гіперпосилання:
    • Використовують повну URL-адресу (наприклад, https://www.example.com/page.html) як адресу посилання.
    • Цілком вказують шлях до файлу або ресурсу незалежно від поточного розташування документа.

    2. <a href="https://www.example.com/page.html">Абсолютне гіперпосилання</a>


  2. Відносні гіперпосилання:
    • Використовують відносний шлях до файлу або ресурсу щодо розташування документа.
    • Можуть бути більш короткими та підтримувати динамічне переміщення документа або проекту без необхідності зміни всіх посилань.

    <a href="relative-page.html">Відносне посилання</a>


Приклад відносного посилання, яке посилається на якір у тому самому документі:

<a href="#section1">Перейти до Розділу 1</a>

Форми HTML надають механізм для збору інформації від користувача. Вони дозволяють користувачам вводити дані та надсилати їх на сервер для обробки. Форми використовуються для різних цілей, таких як вхід на сайт, надсилання коментарів, замовлення товарів та інші взаємодії з користувачем.

Приклад керуючих елементів форми включає різні типи полів введення, кнопки та інші елементи. Ось кілька прикладів:

  1. Текстовое поле:

    2. <label for="username">Им'я користувача:</label>
    <input type="text" id="username" name="username">

  2. Пароль:

    3. <label for="password">Пароль:</label>
    <input type="password" id="password" name="password">

  3. Прапорець (чекбокс):

    4. <input type="checkbox" id="subscribe" name="subscribe" value="yes">
    <label for="subscribe">Підписатися на розсилку</label>

  4. Радіокнопка:

    5. <input type="radio" id="male" name="gender" value="male">
    <label for="male">Чоловічий</label>

    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Жіночий</label>

  5. Випадаючий список:

    6. <label for="country">Місто:</label>
    <select id="country" name="country">
    <option value="kyiv">Київ</option>
    <option value="kharkiv">Харків</option>
    <option value="lviv">Львів</option>
    </select>

  6. Кнопка відправлення:

    7. <input type="submit" value="Відправити">

Ці елементи форми дозволяють користувачам вводити текст, вибирати опції, встановлювати прапорці та радіокнопки, а також надсилати дані на сервер.

Селектор класу (.) використовується для стилізації елементів із певним класом. Можна використовувати кілька елементів на сторінці. Позначається точкою перед ім'ям класу.

Приклад:

.my-class {
color: blue;
}

В HTML:

<p class="my-class">Цей текст буде синього кольору.</p>

Селектор ідентифікатора (#) використовується для стилізації елемента з унікальним ідентифікатором. Має бути унікальним на сторінці. Позначається решіткою перед ім'ям ідентифікатора.

Приклад:

#my-id {
font-weight: bold;
}

В HTML:

<p id="my-id">Цей текст відображатиметься жирним шрифтом.</p>

Селектори псевдоелементів дозволяють стилізувати певні частини елементів, такі як перша літера, перший рядок, або елементи, які знаходяться в певних станах (наприклад, при наведенні курсору).

Приклади:

  1. Перша літера (::first-letter): Застосовує стилі до першої літери у кожному абзаці.

    2. p::first-letter {
    font-size: 150%;
    color: red;
    }

  2. Перший рядок (::first-line): Застосовує стилі першого рядка в кожному абзаці.

    3. p::first-line {
    font-weight: bold;
    }

  3. Стан наведення (:hover): Застосовує стилі для посилань при наведенні курсору.

    4. a:hover {
    text-decoration: underline;
    }


Ці селектори додають додаткові можливості для точкового керування стилями у веб-розробці.
